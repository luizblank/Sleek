<p>A fully native iOS application built with SwiftUI designed to help users organize and track their personal wishlists. The project focuses on a polished UI/UX and deep system integration.</p>

<p><b>Key Technical Highlights:</b></p>
<ul>
  <li><b>Share Extension</b>: Implemented a custom extension allowing users to add items directly from Safari or other apps without opening the main application.</li>
  <li><b>Animations & UI</b>: Added custom animations to make your experience dynamic.</li>
  <li><b>Localization</b>: Localized support (including Portuguese) to handle multi-language users.</li>
  <li><b>Tech Stack</b>: SwiftUI, UserDefaults/CoreData and Share Sheet API.</li>
</ul>
